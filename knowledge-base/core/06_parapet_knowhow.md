# 地覆壁高欄の知見

地覆・壁高欄（parapet_curb_and_barrier）のJSON作成時に必要な専門知識。型知で最も頻繁に使われる構造物。

---

## 地覆・壁高欄が最頻出な理由

橋梁補修工事では、ほぼ毎回地覆・壁高欄の打替えが発生する。橋梁の端部にある構造物なので、凍害・塩害・車両衝突による損傷を受けやすい。石岡組の核心業務。

---

## 地覆の型枠構成

### L字型3面型枠

```
       壁高欄（Phase 2で後打ち）
      ┌───┐
  ────┤   ├────
  │ 地覆   │  │
  │ t=350  │  │  ← 3面型枠（外面＋内面＋天端）
  ├────────┴──┤
  │  底版部    │  ← 床版上面が打継ぎ面
  ════════════ ← 床版
```

| 面 | セパ | 仕上げ |
|-----|------|--------|
| 外面（車道側） | B型 or C型 | 打放し or 埋設 |
| 内面（外側） | B型 or C型 | 打放し |
| 天端 | なし | コテ仕上げ or 型枠 |

### コンパネの割付

- 地覆高さ350mm＋余裕50mm → 400mm → 横使い（900→400mmカット）
- **端材（500mm）は天端型枠（350mm幅）に転用可能** → 材料ロス削減
- 延長方向は1800mmで横使い → 端数をカット材で処理
- セパは1段のみ（高さ400mm以下のため）

## 壁高欄の型枠構成

### 薄壁2面型枠

```
    ←200〜300mm→
    ┌──────────┐
    │ 壁高欄   │ H=800〜1100mm
    │ 2面型枠  │
    │ B型セパ  │ ← 打放し面のためPコン使用
    └──────────┘
    ── 地覆天端 ──
```

| 面 | セパ | 仕上げ |
|-----|------|--------|
| 車道側 | B型（●） | **打放し**（走行車両から見える） |
| 外側 | B型 or C型 | 打放し |

### コンパネの割付

- 壁高1000mm → 縦使い（1800→1050mmカット）
- 延長方向は900mmで並べる
- セパは3段（下段150mm、中段500mm、上段850mm）
- **中央割（美観のため）** — 打放し面では左右対称の割付が望ましい

## AB-セパ干渉チェック（★最重要機能）

壁高欄にはアンカーボルト（AB）が@2000mmで埋設される。これとセパレーターが干渉する。

### チェックロジック

```
1. AB位置を座標リストとして取得（@2000mm等）
2. セパ位置をピッチから自動生成
3. 全ABについて最近接セパとの距離を計算
4. 距離50mm未満 → ★警告
```

### 干渉した場合の対策

- セパ位置をずらす（AB近傍だけピッチを変更）
- AB近傍のセパを省略し、両隣のセパを太径に変更
- ABの配置変更を発注者に協議（最終手段）

## 目地の重要性

### 地覆と壁高欄の目地位置一致ルール（★最重要）

```
壁高欄  │目│        │目│
────────┤地├────────┤地├──
地覆    │ │        │ │
────────┤位├────────┤位├──
        │置│        │置│
        └─┘        └─┘
        ↑          ↑
        必ず同一位置！ずれると水が入る
```

- 伸縮目地間隔: @10〜15m
- 目地材: 発泡スチロール t=20mm
- 止水板: 塩ビCC200×5（目地中央）
- シーリング: ポリウレタン系 W20×D15

### JSON作成時の目地処理

```
Step 1: 目地間隔を決定（@10m等）
Step 2: 全延長を目地間隔で分割
Step 3: 1区間内の寸法を算出（10,000mm - 20mm = 9,980mm）
Step 4: 1区間内でコンパネを割付
```

**目地が割付の起点。目地なしで割付を始めてはいけない。**

## 連続打設の留意点

地覆は橋軸方向に長い構造物（20m以上）を連続打設することが多い。

- 打設速度の管理（流し込みすぎると未充填部分が発生）
- バイブレーターの挿入間隔（@50cm以下）
- 薄い部材なのでバイブレーターで型枠を叩くと変形する
- 天端のレベル管理（天端定規を使用）

---

## JSON作成時の注意

- `casting_method` で別打ち/一体打ちを指定
- `anchor_bolts.positions` にAB位置の座標リストを入れる
- `interference_check` を壁高欄の面に設定（min_clearance_mm: 50）
- 地覆と壁高欄の `joints.expansion_joint_interval_mm` は同じ値にする
- 壁高欄のセパは`layout_method: "中央割"` を推奨（美観のため）
