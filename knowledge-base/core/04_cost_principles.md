# 原価管理の原則

型枠工事の原価管理に関する知識。石岡組の経営哲学（阿座上洋吉先生の理論）に基づく。

---

## 浮遊原価の概念

阿座上洋吉先生（石岡組の恩師）が提唱する原価管理理論の核心。

### 浮遊原価とは

製造業では原価は「材料費＋労務費＋経費」で確定する。しかし建設業では**同じ作業でも現場条件によって原価が大きく変動する**。この変動する部分が「浮遊原価」。

### 型枠工事における浮遊原価の例

| 要素 | 固定的な原価 | 浮遊する原価 |
|------|------------|-------------|
| コンパネ | 購入単価 | 転用回数（1回で捨てるか、5回使うか） |
| セパレーター | 購入単価 | 壁厚に応じた長さの変動 |
| 大工工数 | 標準歩掛 | 現場のアクセス・天候・技量による変動 |
| 支保工 | リース単価 | 設置期間（養生日数に依存） |
| 加工時間 | — | 割付計画の良否で大幅に変わる |

### 型知で浮遊原価を抑制する

- **割付図の事前作成** → 現場でのカット作業を最小化 → 加工時間の短縮
- **端材転用の計画** → コンパネの無駄を削減（例：地覆外面の端材→天端に転用）
- **正確な数量表** → 材料の過不足を防止 → 余剰在庫・追加発注のコスト削減
- **セパの最適選定** → 太径セパの不必要な使用を回避

## 転用回数と単価

コンパネの1枚あたり単価（12mm t、900×1800）：

| 転用回数 | 実質単価の考え方 |
|---------|----------------|
| 1回（使い捨て） | 購入単価のまま |
| 3回転用 | 購入単価 ÷ 3 |
| 5回転用 | 購入単価 ÷ 5 |

**転用回数を増やすための条件：**
- 脱型を丁寧に行う（テコで無理にこじらない）
- 剥離剤を確実に塗る
- 釘穴を最小限にする
- 保管時に雨に濡らさない

## 型枠工事の歩掛

### 標準歩掛（国交省積算基準の目安）

| 構造物 | 型枠面積あたりの歩掛 | 備考 |
|--------|---------------------|------|
| 基礎 | 0.5〜0.7人工/m² | 単純な箱型 |
| 擁壁（竪壁） | 0.6〜0.8人工/m² | 高さに比例して難度UP |
| 橋台 | 0.7〜1.0人工/m² | 多部位・テーパーあり |
| 橋脚 | 0.8〜1.2人工/m² | 高所作業・多段リフト |
| BOXカルバート | 0.7〜0.9人工/m² | 内外の二重型枠 |
| 地覆・壁高欄 | 0.8〜1.0人工/m² | 薄壁・AB干渉 |
| 床版（底型枠） | 0.6〜0.8人工/m² | 支保工込み |

### 型知で歩掛を改善できる部分

- 加工時間の短縮（事前に割付図があるから迷わない）
- 建込み時間の短縮（番付が振ってあるから順番がわかる）
- 手戻りの削減（3Dで事前確認できるから抜け漏れがない）

## 設計変更で利益を確保する

型枠の原価管理で最も効果が大きいのは**設計変更による追加費用の確保**。

### 事例：目地の追加提案
- 当初設計に目地がない → 温度ひび割れのリスク
- 技術提案で目地＋膨張材を追加 → 品質向上
- wise社事例では**720万円増（直工）**の設計変更を勝ち取った

### 型知から設計変更につなげる流れ
1. 型知で割付を検討中に「目地がない」「間隔が広すぎる」に気づく
2. くろたんと壁打ちで根拠を整理
3. 協議資料を作成して発注者に提案
4. 設計変更 → 追加費用確保
