# ベテラン表現変換辞書

現場のベテラン大工・現場監督が使う言葉と、若手やAI向けの解説。現場代理人との壁打ちで出てくる用語を正確に理解するために。

---

## 型枠関連

| ベテラン表現 | 正式名称・意味 |
|-------------|-------------|
| セキ板（せきいた） | コンクリート型枠用合板（コンパネ）のこと。堰板 |
| パネコート | 片面塗装合板（コンパネの上位品）。打放し面に使用 |
| ハナ | 型枠の端部。「ハナを合わせる」＝端部の位置を殃確に合わせる |
| ハナをつぶす | 型枠端部の角を面取りする。面木を当てること |
| アゴ | 型枠の下端。「アゴをとる」＝下端の位置を決める |
| 根巻き（ねまき） | 型枠下端と均しコンの隙間をモルタルで塞ぐこと |
| ノロ | セメントペースト。型枠の隙間から漏れ出るもの。「ノロが吹く」 |
| 桟木（さんぎ） | 30×60mmの角材。コンパネの裏に打ち付けて補強 |
| 端太（ばた） | 型枠を外側から押さえる角材（端太角）or 鋼管（端太パイプ） |
| 控え（ひかえ） | 型枠を外側から支える斜めの支柱。単管パイプ＋クランプ |
| 面木（めんぎ） | 15×15mmの三角形の棒。コンクリートの角を面取りするため |
| 番付（ばんづけ） | コンパネに付ける番号。A-1-01のように面・段・列で表記 |
| 片追い（かたおい） | 型枠の割付方法。一端から順に並べて最後の端数を調整 |
| 中央割（ちゅうおうわり） | 中央から左右に均等に割り付ける方法。打放し面の美観用 |

## セパレーター関連

| ベデラン表現 | 正式名称・意味 |
|-------------|-------------|
| セパ | セパレーター。対面の型枠間隔を保持する鋼棒 |
| 2分5厘（にぶごりん） | セパの太さ。W5/16インチ＝φ8mm |
| 3分（さんぶ） | セパの太さ。W3/8インチ＝φ9.5mm |
| Pコン | プラスチックコーン。B型セパの端部に付けるプラ部品 |
| フォームタイ | セパの端をナットで締めるための金具 |
| 座金（ざがね） | フォームタイの下に入れるワッシャー |
| B型セパ | Pコン付きセパ。仕上げ面用。脱型後にPコン穴が残る |
| C型セパ | ナット式セパ。埋戻し面用。ナットを外すだけで脱型 |

## コンクリート関連

| ベテラン表現 | 正式名称・意味 |
|-------------|-------------|
| 打つ（うつ） | コンクリートを打設すること。「今日打つ」＝今日コンクリート打設 |
| 回す（まわす） | コンクリートをポンプ車で圧送すること。「回してくれ」 |
| 締める（しめる） | バイブレーターで振動を与えてコンクリートを締め固めること |
| レイタンス | コンクリート表面に浮き出る薄い脆弱層。打継ぎ前に除去必須 |
| ジャンカ | 豆板。コンクリートの充填不良で粗骨材が露出した部分 |
| コールドジョイント | 打設の間隔が空きすぎて一体化しなかった打継ぎ面 |
| パンク | 型枠がコンクリートの側圧に負けて変形・破損すること |
| ノロ吹き | 型枠の隙間からセメントペーストが漏れ出ること |
| グリーンカット | コンクリートが半硬化の段階でレイタンスを除去する方法 |
| マスコン | マスコンクリート。部材厚が大きく温度ひび割れリスクが高い |

## 施工関連

| ベテラン表現 | 正式名称・意味 |
|-------------|-------------|
| 墨を出す | 墨出し。構造物の位置を地面や均しコンに線で示す |
| 建てる（たてる） | 型枠を組み立てること。「型枠を建てる」 |
| 返す（かえす） | 型枠を転用すること。「パネルを返す」＝脱型して次の場所に使う |
| ばらす | 型枠を解体すること。脱型と同義 |
| 起こす（おこす） | 傾いた型枠を垂直に修正すること |
| 引く（ひく） | 型枠にチェーンやワイヤーを掛けて引っ張ること（倒壊防止） |
| 上げ越し（あげこし） | キャンバー。底型枠を打設後のたわみを見越して高めに設置 |
| 養生（ようじょう） | コンクリート打設後の保護・水分管理 |

## この辞書の使い方

現場代理人との壁打ちで「ハナを合わせてから建てたい」と言われたら、「型枠端部の位置を正確に合わせてから組み立てたい」と理解する。JSONの `notes` に「端部の位置合わせに注意」と記載する。

ベテランの言葉を正確に理解してJSONに反映することが、型知の品質を決める。
