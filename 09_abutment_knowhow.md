# 橋台型枠の知見

橋台（abutment）のJSON作成時に必要な専門知識。橋台は多部位・多フェーズで最も複雑な構造物。

---

## 橋台の部位構成

```
          ┌─ パラペット（Phase 4）─┐ ← 支承台精度±2mm
          │  H=800                │
  ┌───────┤                      ├───────┐
  │       │                      │       │
  │ 翼壁  │    竪壁（Phase 2-3）  │ 翼壁  │
  │(Ph5)  │    H=5000            │(Ph5)  │
  │       │    テーパーあり       │       │
  │       │                      │       │
  ├───────┴──────────────────────┴───────┤
  │       フーチング（Phase 1）          │
  │       t=1500  B=6000                 │ ← マスコン注意
  └──────────────────────────────────────┘
```

---

## フーチングの型枠

### 特徴
- 大型（幅6m×奥行4m×高さ1.5m等）
- **セパは使えない**（幅が広すぎて対面セパが座屈すク）
- 控え杭方式（H-200型鋼 or 木杭で外から押さえク）
- 4面の箱型型枠

### マスコンクリート判定
- **フーチング厚1m以上 → マスコンクリートに該当する可能性大**
- 温度応力解析が必要になる場合がある
- JSONの `notes` に「マスコンクリート対策の検討が必要」と記載
- wise社事例: 橋台工で温度応力解析→誘発目地＋膨張材→720万円増

## 竪壁の型枠

### リフト分割
- 竪壁高さ5m → 1リフト2.5mで2リフト
- 目安: 1リフト2.5〜3m
- JSONの `stem.lift_height_mm` と `stem.lift_count` で指定

### 下部セパの太径切替
- 竪壁の下部（特に1リフト目）は側圧が最大
- 側圧計算: P = 24 × H = 24 × 2.5 = 60kN/m²
- W5/16の許容引張力14kN → 負担面積0.23m² → ピッチ480mm → ギリギリ
- **安全側でW3/8を推奨**（許容引張力20kN → ピッチ577mm → 余裕あり）
- JSONの `separators` で段ごとに `diameter` を切り替え可能

### テーパー（背面の傾斜）
- 竪壁はしばしば背面にテーパーがある（底部1200mm→天竿800mm等）
- 擁壁の勾配セパ長と同じロジック → 段ごとにセパ長が変わる
- JSONの `stem.taper: true` と `thickness_top_mm` / `thickness_bottom_mm` で指定

## パラペットの型枠

### 支找台の精度管理（★最重要）
- 支找（橋座）を載せる台座の部分
- **レベル精度±2mm** — 橋台型枠で最も厳しい精度要求
- 天端定規（鋷製L=2000mm）で管理
- JSONの `parapet.bearing_seat.level_tolerance_mm: 2` で明記

### 支找アンカーボルト
- M30〜M42の太いAB
- 位置精度±5mm
- テンプレート（鋼製定規）でABの位置を固定して打設
- JSONの `notes` に精度要求を記載

---

## 翼壁の型枠

### テーパー形状
- 翼壁は竪壁から離れるにつれて高さが低くなる（台形 or 三角形）
- コンパネの天端を斜めカットする必要がある
- JSONの `wing_wall.height_start_mm` と `height_end_mm` で始端・終端の高さを指定

### 竪壁との接続部
- 鉛直打継ぎ面になる
- 差し筋の処理が必要
- 止水板の設置を検討

---

## 各部位間の打継ぎ

| 打継ぎ | 方向 | 処理 | 止水板 |
|--------|------|------|--------|
| フーチング→竪壁 | 水平 | チッピング＋レイタンス除去 | 必要（PVC CC200） |
| 竪壁リフト間 | 水平 | チッピング＋レイタンス除去 | 場合による |
| 竪壁→パラペット | 水平 | チッピング＋レイタンス除去 | 場合による |
| 竪壁→翼壁 | 鉛直 | チッピング＋差し筋 | 場合による |

## 温度応力解析と設計変更

### knowledge-base内の参考資料
`10_joint_design.md`（wise社事例）を参照。

### 橋台での検討フロー
1. フーチング厚1m以上 → マスコン該当の可能性
2. 竪壁のL/H比が大きい → 外部拘束度が大きい → ひび割れリスク高
3. 当初設計に誘発目地がない → 技術提案の機会
4. 温度応力解析（有限要素法）で5ケース比較
5. 最適案（誘発目地＋膨張材等）を選定
6. 発注者に協議 → 設計変更 → 追加費用確保

### JSON作成時の注意
- `notes` に「マスコンクリートに該当する可能性がある」と記載
- フーチング厚1.5m以上なら必ず言及
- ひでさんに相談 → くろたんと壁打ちで根拠を整理 → 協議資料作成
